التقنيات المستخدمة
- PHP 
- MySQL
- javascrpit,css
- XAMPP لتشغيل السيرفر المحلي
. طريقة التشغيل
1. نزّل المشروع على جهازك.
2. افتح XAMPP وشغّل Apache و MySQL.
3. استورد قاعدة البيانات database.sql من مجلد المشروع إلى phpMyAdmin.
4. ضع المشروع في مجلد htdocs داخل XAMPP.
5. افتح المتصفح واذهب إلى:
   http://localhost/project_folder/index.php
6. سجل الدخول باستخدام الحساب:(كمدير)
   - Username: admin
   - Password: admin
  
   
فيني سجل دخول كمدير ودكتور وموظف كل منهم باستخدام username وpassword مختلفيين
